# Student Results Analyzer

## What it Does

This is a simple Python console application that analyzes student grades. It calculates each student's total and average score, then finds the student with the highest total score and the student with the lowest total score.

## How to Run It

1. Make sure Python 3 is installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/Belalmohamed448/Student-Results-Analyzer.git
   ```
3. Open the project folder.
4. Run the program:
   ```bash
   python student_results_analyzer.py
   ```
